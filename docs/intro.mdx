# Selamat Datang di Knowledge Hub

Ini adalah paragraf pertama dari dokumenmu. Kamu bisa menuliskan berbagai informasi mengenai arsitektur data di sini.

## Fitur Utama

- **Pencarian Cepat:** Temukan data dalam hitungan detik.
- **Integrasi Mudah:** Mendukung format `.md` dan `.mdx`.

> **Catatan:** Jangan lupa untuk menjalankan `npm run build` jika ingin mengetes fitur pencarian lokal!
