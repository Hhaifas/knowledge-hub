# Ikhtisar Proyek: Otomatisasi Notulensi Rapat

Dokumen ini menguraikan arsitektur utama untuk sistem otomatisasi pembuatan _Meeting Minutes_. Sistem ini dirancang untuk mendengarkan audio rapat, mentranskripsikannya, dan menghasilkan ringkasan yang terstruktur.

## Objektif

Mengurangi waktu manual dalam pembuatan notulensi dan pelacakan tugas (_task tracking_) setelah rapat selesai.

## Teknologi Utama

- **Speech-to-Text:** Whisper API
- **LLM Engine:** Azure OpenAI
- **Orkestrasi:** LangGraph
